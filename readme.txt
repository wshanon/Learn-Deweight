exchange.py为xml转为txt文件，将xml的坐标提取出来进行归一化，保存在txt文件中，
exchange2.py为xml转为txt文件，但没有进行归一化，（分辨率的width，longth，xmin,xmax,ymin,ymax）保存在txt中
deweight.py为去重文件，并将去重的文件保存在新的路径下，其中第一个文件没有进行去重，保存在新的路径下
文件中，其中，包含用于可视化的文件（与去重的内容相同）保存在savevisualtxt中，
对于去重的方法：1.数据集进行提取，
              2.一种使用数据之间进行相减，精确度小于0.015，进行保存，在使用列表将数据遍历，将其去重，保存去重之后的数据
              另一种使用IOU（交并比）进行去重，用面积比大于0.9，进行保存，在使用列表将数据遍历，将其去重，保存去重之后的数据
dtest.py为deweight.py的尝试运行版
test.py为进行可视化的，用于去重之后，前后对的可视化，
txt-xml.py为txt转化为xml文件
main.py将所有的py文件进行运行
